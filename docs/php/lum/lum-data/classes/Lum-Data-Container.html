Attempt to detect the format of data, and return a simple string for it.

The default version supports the following formats:

array => 'array' SimpleDOM => 'simple_dom' SimpleXMLElement => 'simple_xml' DOMNode => 'dom_node'

string => see detect_strinc_type()

If you use this in a base class, then sub-classes can override it and extend it with additional supported formats.
