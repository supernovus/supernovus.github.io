Class: Base

@lumjs/tests/test~Base(optsopt)

Minimalistic base class for the Test library.

This doesn't have any of the regular testing methods, it's simply the core properties, constructor, and output methods.

done()

We're done testing.

This will mark the test-set as finished, so attempting to run further tests after will result in a RangeError being thrown.

If no Harness is in use, this will also run this.output().
