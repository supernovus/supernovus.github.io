Functional

A new test instance and a set of functions wrapping it.

Type:
  • object
Properties:
Name Type Description
test Test

The new test instance.

All of the rest of the properties are functions that can be imported into a JS scope using destructuring, and which wrap the corresponding test instance method.

Basically anything that is a method of the Test instance will become a wrapped function available in this structure.
