functional(optsopt) → {Functional}

Create a function that will export a bunch of wrapper functions that can be imported via object destructuring.

Usage is like:

const {plan,ok,isa} = require('@lumjs/tests').functional({module});

plan(2);
ok(true, 'ok() works');
isa(isa, 'function', 'isa is a function');

Because each time this function is run it creates a new test instance, and a new set of wrapper functions for the test instance, you could create multiple sets of functional tests in their own private scopes. Not sure why you'd want to do that, but in case you do, there ya go.
