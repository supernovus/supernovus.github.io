Module: @lumjs/tests/harness/parser

A simplistic TAP parser.

Currently only handles very basic TAP v12 output. This is loosely based on the TAP parser in my PHP Lum\Test\Harness library, but is split off into its own class, and how it handles the parsed data is quite different. I think I may eventually update the PHP version to be more like this one.
