setDOM(optsopt) → {void}

Set the dom instance.

Sets this.dom to an instance of the @lumjs/dom helper library (referred internally as LumDOM). It has several ways of being called to provide a flexible way to quickly change the instance.

Parameters:
Name Type Attributes Description
opts object | string <optional>

Many ways to get the dom instance.

  • If opts is a string it's assumed to be an HTML document, and will be passed to getLumDOM() to create a new dom instance.
  • If opts is a LumDOM (@lumjs/dom) instance, it will be assigned directly to this.dom with no further action required.
  • If opts is a Window instance, it will be passed to getLumDOM() to get the dom.
  • If opts is a plain object, it supports the same parameters as the constructor with the same preferences.
Properties
Name Type Attributes Description
dom LumDOM <optional>

A @lumjs/dom instance.

If specified, directly assign the dom instance. No further options are used if this was set.
window Window <optional>

A window object.

This will be passed to getLumDOM() to build a new instance.
doc string <optional>

An HTML document to parse.

This will be passed to getLumDOM() to build a new instance.
domOptions object <optional>

LumDOM constructor options.

If specified, this will be passed to getLumDOM() as the constructor options.
