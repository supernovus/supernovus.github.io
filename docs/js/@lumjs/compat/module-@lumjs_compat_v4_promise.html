Build a Promise object with Deferred compatibility.

options object | boolean <optional>

Options for the Promise object.

If options is a boolean, it's assumed to be the jquery option.

Properties
Name Type Attributes Description
jquery boolean <optional>

Use jQuery.Deferred API?

  • If true, use this._extendJQuery(options) to define methods.
  • If false, use this._extendInternal(options) to define methods.
Deprecated:
  • Use the Javascript `Promise` object directly. If you need compatibility with the `jQuery.Deferred` API, just use the `jQuery.Deferred` API itself. If you need the `deferDone` or `deferFail` methods, use this until the `@lumjs/defer` library is released.
