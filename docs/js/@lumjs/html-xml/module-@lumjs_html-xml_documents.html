Module: @lumjs/html-xml/documents

The full sample document module set.

This is probably more than you really need. The default set is probably fine for most tests. If you really want everything, requiring this module is the way to go.

Note:

For every lowercase property defined directly in this module, there is an uppercase version as well.

So for example, the default property can also be referenced as DEFAULT.
