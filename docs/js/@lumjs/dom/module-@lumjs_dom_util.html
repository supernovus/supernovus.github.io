(static) HTML_MODE

An Enum of modes for the html() method.

Properties:
Name Type Description
AUTO number

Auto-determine the return value.

  • If there are any child elements in the <head/>, or there are no child elements in the <body/>, then the top-level <html/> element will be returned.
  • If the <body/> has only one child element, then the single child element will be returned.
  • If there are multiple elements in the body, return the body.children property which is an HTMLCollection of child elements.
AUTO_BODY number

Auto-determine the return value.

This is the same as the AUTO mode, except that if there are multiple elements in the body, this mode returns the <body/> element itself.
DOC number

Return the document object itself.
HTML number

Return the <html/> element.
BODY number

Return the <body/> element.

If there is no body element, this mode will return null.
