(static) has(ns) → {boolean}

See if a root-level name is defined.

This adds a context.has.{name} boolean property which caches the result so it can be referred to directly.

In any JS environment with the Proxy object (which honestly should be all modern ones), you can simple do context.has.SomeObject instead of context.has('SomeObject') and it will do the Right Thingâ„¢.
