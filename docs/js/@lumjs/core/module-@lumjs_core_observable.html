Make an object support the Observable API.

Adds on(), off(), one(), and trigger() methods.

Properties
Name Type Attributes Default Description
wildcard string <optional>
 '*'

The event name used as a wildcard.
wrapthis boolean <optional>
 false

If true, this will be a wrapper.

If 'wrapthis' is true, the function will be called with a wrapper object as the 'this' variable instead of the target object. The wrapper will be:

{
  self: el,        // The target object.
  name: event,     // The event name that was triggered.
  wildcard: bool,  // Will be true if this was a wildcard event handler.
  func: function,  // The function being called.
}
addname boolean <optional>
 !opts.wrapthis

If true callbacks with multiple events will have the name of the triggered event added as the first parameter.
addis boolean <optional>
 opts.wrapthis

If true add immutable property named isObservable which will have a value of true.
addme string <optional>

If set, add a method with this name to the object, which is a version of observable() with the default options being the same as the current opts. If opts.wrapthis is true, then this defaults to 'makeObservable'. In any other case it defaults to null.
